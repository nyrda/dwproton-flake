# Upstream

This repository is a fork of the DW-Proton Nix flake originally maintained at:

https://github.com/imaviso/dwproton-flake

DWProton releases are published by Dawn Winery at:

https://dawn.wine/dawn-winery/dwproton/releases

The scheduled GitHub Actions workflow updates `default.nix` from the latest Dawn Winery release and opens a pull request when the packaged version changes.
